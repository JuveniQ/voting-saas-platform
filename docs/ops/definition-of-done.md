# Definition of Done

Work is `Done` only when all of the following are true:

- the change merged through a pull request
- `npm run lint`, `npm run typecheck`, `npm run test`, and `npm run build` pass
- reviewer approval is complete
- acceptance criteria were checked against the final behavior
- docs were updated if behavior, process, or contracts changed
- screenshots or API evidence are attached when relevant
- preview or local verification was completed for risky changes

For docs-only or planning-only changes, screenshots and API evidence are optional.
