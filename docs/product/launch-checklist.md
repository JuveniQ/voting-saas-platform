# Launch Checklist

- Confirm wildcard DNS and host routing work for platform/admin/tenant surfaces.
- Verify Neon database, migrations, and RLS policies in staging.
- Configure Better Auth secrets and admin enrollment path.
- Configure Resend domain, sender, and OTP deliverability checks.
- Enable Cloudflare cache bucket, images binding, and production worker name.
- Run replay, rate-limit, and burst vote tests.
- Dry-run backup and restore using a staging snapshot/branch.
- Validate audit logs and export approval flow.
- Review privacy copy, campaign rules copy, and support escalation runbook.
